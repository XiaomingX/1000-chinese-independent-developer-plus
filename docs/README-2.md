备注：您当前查看的是主版面，收录的产品是打开即用，和子版面中的产品类型不同。

**1. 为什么有这个表**
作为开发者其实比较好奇其他人在做什么业余项目（不管目的是做到盈利/玩票/试试看）
所以特意建了这个库。欢迎各位开发者把自己的项目加进来~ 发 Pull Request 或 Issue 即可 <br/>
（入选标准：必须是网站或App，不能是开发者工具或论坛型网站）

* :white_check_mark: **[Runway Aleph](https://runwayaleph.net/)**：前沿 AI 视频编辑平台。通过文本指令即可实时调整视频中的光影、生成新视角或应用风格迁移，让非专业人士也能创作专业级视频。
* :white_check_mark: **[AI LRC Generator](https://ailrcgenerator.com/)**：针对音乐爱好者与播客主的痛点，利用 AI 自动识别音频并生成精准的时间轴 LRC 歌词文件。
* :white_check_mark: **[KissPixel](https://kisspixel.ai/)**：集成了换脸、风格转换、图像扩展及放大等六大核心功能的 AI 视觉工作台，生成的图像享有商业许可。

#### 开发者与技术工具

* :white_check_mark: **[EasyTransfer](https://file.ch3nyang.top/)**：免费、匿名的端到端加密（E2EE）文件传输工具。支持内网穿透，只需简单的设备代码即可跨网络快速传文件，安全且无需注册。
* :white_check_mark: **[NextDevKit](https://nextdevkit.com)**：Next.js SaaS 启动模板。预集成了支付、认证、博客及数据库模块，帮助开发者在一天内上线产品。
* :white_check_mark: **[LLMOCR](https://llmocr.com)**：基于大模型的 OCR 服务，相比传统 OCR，能更好地理解复杂布局、表格及手写体。

#### 垂直领域与有趣工具

* :white_check_mark: **[去留助手 AI](https://www.huiguo.info/)**：为海外华人设计的决策工具。基于大模型与行业大数据，评估“回国”与“留洋”的职业发展优劣，并计算同龄竞争力坐标。
* :white_check_mark: **[怼怼侠](https://www.duiduixia.com/)**：利用 AI 优雅且幽默地回击阴阳怪气，不带脏话却杀伤力十足。
* :white_check_mark: **[中文古籍數字復刻計劃](https://github.com/shanleiguang/vBooks)**：专注于古籍刻本的数字化复刻，提供支持圈注、查询的精美电子书，旨在通过沉浸式体验传承文化。
* :white_check_mark: **[晚星 (App)](https://dream.sayhi.cc/)**：极简纯粹的白噪音 App，提供上百种自然音效，支持自定义自由组合，专为助眠与冥想设计。
* :white_check_mark: **[AIRI](https://github.com/moeru-ai/airi)**：AI 虚拟角色容器。由 LLM 驱动，能进行实时语音聊天，甚至能在 Minecraft 和 Factorio 中陪你一起玩游戏。
