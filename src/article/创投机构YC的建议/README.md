## 如何评估一个创意？(The YC Filter)
1.  **它是否让人省钱？** (Save Money)
2.  **它是否让人赚钱？** (Make Money)
3.  **它是否让人省时间？** (Save Time)
如果一个想法不同时满足以上至少一点，**直接丢弃**。

## 一、获客导向型工具 (Growth Tools)
**核心逻辑**：流量太贵了。帮企业搞到便宜流量的工具，永远有人买单。

| 项目名称 | 核心功能 | YC 价值主张 (Value Prop) | 建议技术栈 (2026) | 目标客户 |
| :--- | :--- | :--- | :--- | :--- |
| **中小商家社群裂变工具** | 自动生成带参数的裂变海报，追踪谁带来了新客户，并自动发放奖励（即使是线下小店）。 | **替代昂贵的 SaaS**。让卖炸鸡的大叔也能像瑞幸咖啡一样玩裂变。 | Python (FastAPI) + 微信/WS API | 线下餐饮、零售店 |
| **内容流量关键词挖掘** | 跨平台（抖音/红书）挖掘“蓝海词”——搜索量大但内容少的词。 | **低成本获客**。告诉创作者“今天即使拍个烂视频，带上这个词也能火”。 | Node.js (Puppeteer) + LLM summary | 焦虑的自媒体人 |
| **本地生活“捡漏”雷达** | 聚合周边 3km 内商家的“临期特惠”或“空闲时段折扣”。 | **双赢**。商家清库存，用户薅羊毛。利用地理位置的即时性打破信息差。 | Vue3 + 高德 API + PWA | 价格敏感型消费者 |
| **直播间智能场控** | 自动识别直播间的弹幕问题，用主播的语气自动回复，甚至自动发福袋带节奏。 | **人力替代**。一个 AI 场控顶 3 个运营，24 小时无休。 | Python + 语音转文字 + LLM | 中腰部带货主播 |

## 二、降本导向型工具 (Efficiency Tools)
**核心逻辑**：与其赚更多的钱，不如少花冤枉钱。

| 项目名称 | 核心功能 | YC 价值主张 (Value Prop) | 建议技术栈 (2026) | 目标客户 |
| :--- | :--- | :--- | :--- | :--- |
| **全渠道客服聚合** | 一个后台回复淘宝、抖音、微信、邮件的所有消息。 | **不再切屏**。客服每天切换窗口浪费 2 小时。把这 2 小时还给老板。 | Electron + 聚合聊天 API | 电商多平台卖家 |
| **发票/报销自动填报** | 拍照 -> 识别 -> 填单 -> 自动给老板发审批微信。 | **消除痛苦**。没人喜欢贴发票。把最反人类的工作交给 AI。 | Flutter + OCR API | 小微企业员工 |
| **网站/服务宕机自愈** | 监控到 404 或卡顿，自动尝试重启服务器或清理缓存。 | **夜间安睡**。让独立开发者半夜不用爬起来修 Bug。 | Go (轻量监控) + Shell 脚本 | 个人站长 |

## 三、合规套利型工具 (Resource Hacks)
**核心逻辑**：利用大公司的规则漏洞（Feature），合规地获取资源。

| 项目名称 | 核心功能 | YC 价值主张 (Value Prop) | 建议技术栈 (2026) | 目标客户 |
| :--- | :--- | :--- | :--- | :--- |
| **云资源“零元购”监控** | 监控 AWS/阿里云 的“新用户免费试用”和“闲置实例竞价”。 | **白嫖算力**。帮初创公司每年省下几千刀的服务器费用。 | Python 爬虫 + 邮件提醒 | 穷学生、初创团队 |
| **多平台比价与隐藏券** | 能算出“拼多多百亿补贴 vs 淘宝李佳琦直播间”到底哪个真便宜。 | **极致省钱**。对抗平台的杀熟算法和复杂规则。 | Chrome Extension | 家庭主妇、羊毛党 |
| **免费商用素材聚合** | 聚合 Unsplash, Pexels 等所有 CC0 协议的素材，并做语义搜索。 | **版权保险**。用这个搜图，永远不用担心收到律师函。 | Next.js + 向量搜索 | 设计师、小编 |

## 四、经典创意的 AI 升级 (Classic Reboot)
旧瓶装新酒，好喝又上头。

| 项目名称 | AI 升级点 | 为什么现在做？ |
| :--- | :--- | :--- |
| **视频剪辑** | 现在的剪辑是“时间轴操作”；未来的剪辑是**“我想把这段变得更像电影一点”** (Style Transfer)。 | 创作者想偷懒，AI 能满足。 |
| **图片处理** | 不要做 PS，做**“一键把图里的文字换成英文”** (In-painting OCR)。 | 跨境电商爆发，改图需求巨大。 |
